Тест системы фильтрации эпизодов

Новые функции

  • Метка завершенности эпизода - отображается зеленая метка "Завершен" на карточке эпизода
  • Фильтр по статусу - возможность фильтровать эпизоды по статусу: Новые Доступные Завершенные
  • Автоматическое определение статуса - система автоматически определяет статус эпизода на основе сохранений
  • Комбинированные фильтры - можно комбинировать фильтры по типу, возрасту и статусу

Логика фильтрации

  • Новые - эпизоды, которые разблокированы, не завершены и не имеют сохранений
  • Доступные - эпизоды, которые разблокированы и не завершены (включая те, что имеют сохранения)
  • Завершенные - эпизоды, которые полностью пройдены
  • Все эпизоды - показывает все эпизоды независимо от статуса

Инструкции по тестированию

  1. Запустите игру и перейдите к выбору эпизода
  2. Проверьте, что на карточке эпизода "Обучение" отображается метка "Завершен" (если эпизод пройден)
  3. Попробуйте использовать фильтр "Завершенные" - должен показаться только завершенный эпизод
  4. Попробуйте фильтр "Новые" - должен показаться эпизод, если он не имеет сохранений
  5. Попробуйте фильтр "Доступные" - должен показаться эпизод, если он не завершен
  6. Проверьте комбинирование фильтров (тип + статус + возраст)

Технические детали

  • Используется функция isEpisodeCompleted() для определения завершенности
  • Используется функция getEpisodeSave() для определения наличия сохранений
  • Данные о статусе добавляются при загрузке эпизодов
  • Фильтрация происходит в реальном времени при изменении фильтров
